
# Machine Learning (ML)

**Machine Learning (ML)** is a subset of AI that allows systems to learn patterns from data instead of relying only on fixed rules.

Key idea:
- the system improves by learning from examples
- it can make predictions or decisions on new data
- it is useful when patterns are too complex to define manually

Common uses:
- fraud detection
- forecasting
- recommendation systems
- classification

---

# How machine learning works

A typical ML workflow looks like this:

**Data Collection -> Data Preparation -> Training -> Testing -> Inference -> Monitoring**

What happens:
- data is collected and cleaned
- the model learns patterns during training
- performance is checked during testing
- the trained model is used on new data during inference
- the model is monitored and improved over time
